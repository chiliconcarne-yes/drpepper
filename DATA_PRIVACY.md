# 🕵️ Data Privacy Policy — ChiliCheck

**Last updated:** 2026-04-20

## Overview

ChiliCheck is a client-side web application designed for education and fun. We believe privacy should be the **default ingredient**, not an upsell.

## What Data We Collect

**None.** ChiliCheck does not collect, store, or transmit any personal data to our servers — because we don't operate any backend servers or databases.

## How the App Works

| Action | Where it runs | Data sent to us? |
|--------|--------------|-------------------|
| Browser fingerprint generation | Your browser (FingerprintJS open-source) | ❌ No |
| IP geolocation lookup | ipapi.co public API (third-party) | ❌ No |
| Pepper score calculation | Your browser | ❌ No |
| Raw browser info display | Your browser | ❌ No |

## Third-Party Services

The only external service contacted by ChiliCheck is:

- **ipapi.co** — A public, read-only REST API used to resolve IP addresses to geolocation data (city, country, ISP, ASN). No API key or authentication token is used. Their privacy policy applies to this request: <https://ipapi.co/privacy/>.

## Cookies & Tracking

- 🚫 No tracking cookies
- 🚫 No localStorage profiling or cross-site identifiers
- 🚫 No analytics SDKs (Google Analytics, Mixpanel, etc.)
- 🚫 No third-party advertising scripts
- 🚫 No social media tracking pixels

## Data Sharing

We do not sell, share, or transfer any user data to third parties — because we do not possess any user data.

## GDPR Compliance

ChiliCheck is **GDPR-friendly by design**: there is nothing to delete because nothing is stored. No consent banner is required because no personal data is processed by us.

## Children's Privacy

ChiliCheck does not knowingly collect data from anyone, including children under 13.

## Changes to This Policy

If this policy changes, the update will be reflected in this file on GitHub with an updated date.
